# 习惯星球 — 用户旅程地图

## 旅程 1：新用户首次使用

```
发现 → 打开 → 引导 → 创建习惯 → 首次打卡 → 查看进度 → 回访
  │      │      │        │          │          │         │
  │      │      │        │          │          │         └─ 第2天收到"别断签"动力
  │      │      │        │          │          └─ 看到进度环和 Streak
  │      │      │        │          └─ 点击卡片，勾选 ✓
  │      │      │        └─ 选模板 or 自定义
  │      │      └─ 4 步 Onboarding（可跳过）
  │      └─ 浏览器打开 index.html
  └─ 朋友推荐 / 自己搜索
```

**关键情感曲线**：
- 打开时：好奇 😊
- 引导时：略 impatient（需可跳过）
- 首次打卡：小成就 🎉
- 全部完成：大满足 🎊（confetti 动画）

---

## 旅程 2：老用户每日使用

```
打开 App → 看问候语/名言 → 浏览今日习惯 → 逐个打卡 → (可选)看统计
   │            │                │              │            │
   │            │                │              │            └─ 查看热力图、成就
   │            │                │              └─ 3 秒内完成
   │            │                └─ 看到 Streak 提醒
   │            └─ 每日一句激励
   └─ < 1 秒加载（本地存储）
```

**设计目标**：整个流程 **< 30 秒**

---

## 旅程 3：习惯管理者

```
进入习惯页 → 筛选分类 → 查看数据 → 编辑/删除 → 新建习惯
     │           │          │          │           │
     │           │          │          │           └─ 选模板快速创建
     │           │          │          └─ 二次确认防误删
     │           │          └─ 总打卡/连续/最长
     │           └─ 健康/学习/生活/工作
     └─ 侧边栏 or 底部导航
```

---

## 旅程 4：数据管理者

```
设置页 → 切换主题 → 导出备份 → (换设备) → 导入恢复
   │         │          │                    │
   │         │          │                    └─ 无缝迁移
   │         │          └─ JSON 文件
   │         └─ 深色/浅色
   └─ 数据安全需求
```

---

## 触点与痛点

| 触点 | 用户行为 | 潜在痛点 | 我们的解决方案 |
|------|----------|----------|----------------|
| 首次打开 | 不知道干什么 | 空白页焦虑 | Onboarding + 示例习惯 |
| 今日页 | 打卡 | 操作步骤多 | 单击卡片即打卡 |
| 断签后 | 打开 App | 挫败感 | Streak 从昨天算起，给一次机会 |
| 统计页 | 看进度 | 数据太抽象 | 热力图 + 柱状图 + 成就 |
| 换手机 | 迁移数据 | 丢失记录 | JSON 导出/导入 |

---

## 关键指标与旅程对应

| 旅程阶段 | 追踪指标 |
|----------|----------|
| 首次使用 | 引导完成率、首日创建习惯数 |
| 每日使用 | DAU、日均打卡次数、打卡耗时 |
| 长期留存 | 7 日/30 日留存、平均 Streak |
| 功能深度 | 统计页访问率、成就解锁率 |
