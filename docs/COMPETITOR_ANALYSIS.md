## 3. 详细对比

### 3.1 Habitica

**定位**：游戏化任务管理，把习惯、日常、待办变成 RPG 体验。

**核心功能**：
- 三类任务：Habits（灵活）、Dailies（重复）、To-Dos（一次性）
- RPG 系统：经验值、金币、装备、宠物、副本
- 社交：Party 组队、Guild 公会、共同任务
- 跨平台云同步（账号体系）
- 开源，核心功能免费；订阅 $4.99/月 解锁 cosmetic

**优势**：
- 激励机制极强，适合游戏爱好者
- 社区活跃，长期留存高
- 功能极其丰富，生态成熟

**劣势**：
- 学习曲线陡峭，新手需理解 RPG 概念
- 界面信息密度高，「只想打卡」的用户觉得重
- 移动端功能弱于 Web
- 打卡路径长：找任务 → 点完成 → 可能还有副本动画

**对习惯星球的启示**：
- ✅ 可借鉴：Streak、成就徽章、社交挑战（V3）
- ❌ 不建议照搬：完整 RPG 系统过重
- 💡 差异化：**3 秒打卡** vs Habitica 的复杂任务体系

---

### 3.2 小日常

**定位**：国内流行的可爱风习惯 / 日常记录 App。

**核心功能**（基于公开信息与用户反馈归纳）：
- 习惯打卡 + 心情 / 日记记录
- 精美主题、贴纸、卡片式设计
- 连续打卡天数展示
- 桌面小组件（iOS / Android）
- 部分功能需会员

**优势**：
- 视觉讨喜，尤其受女性用户欢迎
- 本土化好，中文体验流畅
- 小组件降低打开成本

**劣势**：
- 功能偏「记录」而非「数据分析」
- 免费版限制多（主题、习惯数量等）
- 无 Web 版，数据导出有限
- 游戏化程度低，长期激励依赖美观 UI

**对习惯星球的启示**：
- ✅ 可借鉴：每日一句、可爱 Emoji 图标、主屏幕 Widget（PWA）
- ✅ 可借鉴：心情 / 备注与打卡结合（已实现备注）
- 💡 差异化：**免费 + 开源 + 热力图深度统计**

---

### 3.3 Done

**定位**：极简习惯追踪，Apple 设计奖风格。

**核心功能**：
- 一键打卡，界面极简
- GitHub 风格热力图
- 多习惯、多目标（如「每周 3 次」）
- iCloud 同步（仅 Apple 生态）
- 付费买断 / 订阅解锁无限习惯

**优势**：
- 交互极快，几乎零学习成本
- 热力图直观，数据控喜欢
- 设计精美，符合 iOS 原生审美

**劣势**：
- 仅 iOS，无 Android / Web
- 无社交、无游戏化
- 高级功能需付费
- 习惯类型较单一（次数型支持有限）

**对习惯星球的启示**：
- ✅ 已借鉴：热力图、进度环、极简打卡
- ✅ 可加强：Widget、习惯目标「每周 N 次」
- 💡 差异化：**跨平台 Web + 云同步 + 免费**

---

### 3.4 Loop Habit Tracker

**定位**：开源、离线优先的 Android 习惯 App。

**核心功能**：
- 纯习惯追踪，无任务/待办
- 灵活调度（每 N 天、每周特定日）
- 详细统计图表
- 完全离线，数据本地
- 开源（GPL），无账号体系

**优势**：
- 隐私友好，无服务器依赖
- 统计专业（分数算法、趋势）
- 轻量、稳定、无广告

**劣势**：
- 仅 Android
- UI 偏工程师审美
- 无云同步、无多设备
- 无社交与 gamification

**对习惯星球的启示**：
- ✅ 可借鉴：灵活习惯频率（V2）
- ✅ 已覆盖：本地优先 + 可选云同步（比 Loop 更灵活）
- 💡 差异化：**现代 UI + 云同步 + Web 端**

## 5. 用户体验对比

### 5.1 打卡效率（步骤数）

| 产品 | 典型步骤 | 耗时 |
|------|----------|------|
| Habitica | 打开 App → 找 Daily → 勾选 → 等动画 | ~15s |
| 小日常 | 打开 → 点习惯卡片 | ~5s |
| Done | 打开 → 点圆点 | ~3s |
| **习惯星球** | 打开 → 点卡片 | **~3s** |

### 5.2 首次使用体验

| 产品 | 冷启动 | 问题 |
|------|--------|------|
| Habitica | 需注册 + 创建 Avatar | 门槛高 |
| 小日常 | 引导创建习惯 | 较流畅 |
| Done | 直接添加 | 极简 |
| **习惯星球** | Onboarding + 示例习惯 | 已优化 |

### 5.3 视觉风格定位

```
可爱温馨 ←————————————————→ 专业数据
  小日常          习惯星球          Done / Loop
                        ↑
                    Habitica（游戏奇幻风）
```

**习惯星球** 定位：**简洁现代 + 数据可视化**，介于小日常与 Done 之间，用 Indigo 主色传递专注感。

## 6. SWOT 分析（习惯星球）

| | 内容 |
|---|------|
| **S 优势** | 开源免费、Web 跨平台、3 秒打卡、热力图+周报、可选云同步 |
| **W 劣势** | 无原生 App、无推送（Web 限制）、品牌知名度零 |
| **O 机会** | 小日常/Done 无 Web 版；Habitica 太重；国内轻量开源习惯 App 少 |
| **T 威胁** | 苹果自带健康习惯、Notion/飞书模板、大厂免费工具 |

## 7. 战略建议

### 7.1 短期（V1.x → V2）
1. **PWA 增强**：离线可用、添加到主屏幕、推送提醒
2. **灵活频率**：「每周 3 次」「每 2 天」等 Loop 式调度
3. **GitHub Pages 公网部署**：分享链接即可使用

### 7.2 中期（V2 → V3）
1. **轻量社交**：好友 Streak 排行榜（不做 Habitica 式 RPG）
2. **习惯模板市场**：用户分享习惯组合
3. **Apple Watch / 小组件**（需原生壳或 PWA Widget）

### 7.3 长期差异化
> **「最简单的专业习惯分析工具」** — 比 Done 更跨平台，比 Loop 更美观，比 Habitica 更轻，比小日常更数据向。

---

## 8. 结论

| 竞品 | 最值得借鉴 | 习惯星球应避开 |
|------|-----------|---------------|
| Habitica | 成就体系、社交挑战 | 复杂 RPG、高学习成本 |
| 小日常 | 可爱视觉、心情记录 | 过度依赖付费主题 |
| Done | 极简交互、热力图 | 封闭 Apple 生态 |
| Loop | 灵活调度、统计算法 | 工程师审美 |

习惯星球的核心竞争力：**开源 + Web 原生 + 3 秒打卡 + 专业数据可视化 + 可选云同步**，在「轻量」与「专业」之间占据独特位置。
