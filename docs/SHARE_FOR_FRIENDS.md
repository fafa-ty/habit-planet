## 朋友常见问题 FAQ

**Q：要注册吗？**  
A：不用，打开就能用。

**Q：数据会丢吗？**  
A：数据存在浏览器里。换手机或清缓存会丢，建议偶尔在「设置 → 导出数据」备份。

**Q：能和我的手机同步吗？**  
A：可以。你创建「同步空间」，把同步 ID 和密钥发给我，我在另一台设备输入即可（需要同步服务在线）。

**Q：和 Habitica / 小日常比怎么样？**  
A：更轻、更快，3 秒打卡，免费无广告，但功能没那些 App 多，适合「简单记录坚持」。

**Q：打不开怎么办？**  
A：检查网络；若 GitHub Pages 未部署，可先用本地版 `http://你的电脑IP:8766`（需同一 WiFi）。
