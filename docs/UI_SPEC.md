# 习惯星球 · Figma 线框图 & UI 规范

> 可直接按此文档在 Figma 中搭建设计稿。所有尺寸基于 **375×812（iPhone）** 和 **1440×900（桌面）** 两套断点。

---

## 0. 设计系统（Design Tokens）

### 0.1 色彩

| Token | 浅色模式 | 深色模式 | 用途 |
|-------|----------|----------|------|
| `primary/500` | `#6366F1` | `#818CF8` | 主按钮、进度环、选中态 |
| `primary/100` | `#EEF2FF` | `#312E81` | 选中背景、成就解锁 |
| `success/500` | `#10B981` | `#34D399` | 完成态、Streak |
| `warning/500` | `#F59E0B` | `#FBBF24` | 提醒横幅 |
| `danger/500` | `#EF4444` | `#F87171` | 删除、错误 |
| `bg/base` | `#F8FAFC` | `#0F172A` | 页面背景 |
| `bg/elevated` | `#FFFFFF` | `#1E293B` | 卡片、模态框 |
| `border/default` | `#E2E8F0` | `#334155` | 卡片边框 |
| `text/primary` | `#0F172A` | `#F1F5F9` | 标题 |
| `text/secondary` | `#64748B` | `#94A3B8` | 副文案 |
| `text/muted` | `#94A3B8` | `#64748B` | 辅助信息 |

### 0.2 字体

| 层级 | 字号 | 字重 | 行高 | 用途 |
|------|------|------|------|------|
| H1 | 28px / 22px* | 700 | 1.2 | 页面标题 |
| H2 | 18px | 600 | 1.3 | 区块标题 |
| Body | 16px | 400 | 1.5 | 正文 |
| Caption | 14px | 400 | 1.4 | 副信息 |
| Micro | 12px | 500 | 1.3 | 标签、徽章 |

*移动端 H1 为 22px

**字体栈**：`-apple-system, PingFang SC, Hiragino Sans GB, Microsoft YaHei, sans-serif`

### 0.3 圆角 & 间距

| Token | 值 | 用途 |
|-------|-----|------|
| `radius/sm` | 8px | 输入框、小按钮 |
| `radius/md` | 12px | 卡片 |
| `radius/lg` | 16px | 面板、模态框 |
| `radius/full` | 9999px | 胶囊按钮、Toggle |
| `space/page` | 16px / 32px | 页面边距（移动/桌面） |
| `space/card` | 16px | 卡片内边距 |
| `space/gap` | 10px | 列表项间距 |

### 0.4 阴影

| 层级 | 值 |
|------|-----|
| `shadow/sm` | `0 1px 2px rgba(0,0,0,0.05)` |
| `shadow/md` | `0 4px 12px rgba(0,0,0,0.08)` |
| `shadow/lg` | `0 8px 24px rgba(0,0,0,0.12)` |

## 8. 响应式断点

| 断点 | 布局变化 |
|------|----------|
| `< 768px` | 隐藏 Sidebar，显示 Bottom Nav，Hero 纵向 |
| `≥ 768px` | 显示 Sidebar 240px，隐藏 Bottom Nav，Hero 横向 |
| `≥ 1024px` | Main content max-width 900px 居中 |
