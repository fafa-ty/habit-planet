## 5 分钟课堂演示脚本

```
[0:00-0:30] 开场
「大家好，我做的是习惯星球——一个帮助用户养成好习惯的 Web App。
 痛点是：人人都想坚持，但现有 App 要么太重（Habitica），要么不跨平台（Done）。」

[0:30-1:30] 竞品
「我对比了 4 款产品（打开 COMPETITOR_ANALYSIS.md）。
 结论：习惯星球定位在『轻量 + 数据可视化 + 跨平台』。」

[1:30-3:00] Demo 演示
1. 打开在线 Demo
2. 展示今日页 → 点卡片打卡 → 看进度环变化
3. 打开统计页 → 热力图、成就
4. 设置页 → 深色模式切换

[3:00-4:00] 设计说明
「PRD 里定义了 MVP 和 V2；UI_SPEC 里有完整组件规范；
 用户旅程覆盖新用户、每日使用、数据管理 4 条路径。」

[4:00-4:30] 技术实现
「Vanilla JS 实现 MVP，React 重构版，Node 云同步，GitHub Pages 部署。」

[4:30-5:00] 总结
「核心指标：3 秒完成打卡。下一步：PWA 离线、灵活频率、轻量社交。」
```
